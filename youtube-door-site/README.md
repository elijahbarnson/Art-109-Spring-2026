# The Only Door That Opens

A small HTML/CSS/JavaScript website artwork about excessive screen use.

## How to run

1. Unzip the folder.
2. Open `index.html` in your browser.
3. Hover over apps and click them.
4. Every app errors except YouTube, which redirects to youtube.com.

## Files

- `index.html` — page structure
- `styles.css` — phone and animation styling
- `script.js` — app click behavior
